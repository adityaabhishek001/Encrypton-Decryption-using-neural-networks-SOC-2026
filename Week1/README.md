# Week 1
This week consisted of deep diving into Neural Networks and to create a model that takes inputs of sepa and petal lengths and separations a inputs and makes prediction on the type of flowers.

This was achieved by making a fully connected neural network of size : 4 -> 6 -> 3

Before that the data was cleaned up using Pandas Library. This is a small model on a small dataset of 150 entries that was divided in the 9:1 ratio.

**Hyper Parameters of the Model:**

- Learning Rate: 0.001
- Number of epochs: 15
- Loss : Cross Entropy Loss
- Optimizer : Adam

The Model performed decent regardless of the very tiny datasset and got an final Accuracy of 94.07%.
