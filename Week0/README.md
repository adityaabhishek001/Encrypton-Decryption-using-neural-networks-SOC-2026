## Part 1 - Python and it's Libraries
This week we learnt about the libraries that are used in machine learning like numpy, pandas and matplotlib.
The assignment also consisted of questions on the libraries

- **Pandas** : A library used for loading, viewing, cleaning and manipulating data using DataFrames.
- **NumPy**  : A numerical computing library that provides fast multi-dimensional arrays and mathematical operations for data processing.
- **PyTorch** : An deep learning framework used to build, train, and evaluate neural network models efficiently and easily withoutwriting long lines of code.
- **MatPlotLib** : A library for visualising Data using different kinds of graphs.
